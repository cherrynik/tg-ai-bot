# Инструкция по настройке бота для работы в группах

## ⚠️ ВАЖНО: Отключение Privacy Mode

По умолчанию Telegram боты **НЕ получают сообщения** в группах (Privacy Mode включен). 
Для работы бота в группах нужно отключить Privacy Mode:

### Шаги:

1. Откройте Telegram и напишите **@BotFather**
2. Отправьте команду `/mybots`
3. Выберите вашего бота из списка
4. Выберите **Bot Settings**
5. Выберите **Group Privacy**
6. Выберите **Turn OFF** (отключить)
7. **Перезапустите бота** после изменения настроек

### Проверка работы:

После отключения Privacy Mode:
1. Перезапустите бота: `yarn start` или `npm start`
2. Добавьте бота в группу (если еще не добавлен)
3. Упомяните бота в группе: `@username_бота привет`
4. Бот должен ответить

### Если бот все еще не отвечает:

1. Проверьте, что бот запущен (в консоли должно быть сообщение о запуске)
2. Проверьте, что бот добавлен в группу
3. Проверьте логи в консоли - должны появляться сообщения при получении сообщений
4. Убедитесь, что Privacy Mode действительно отключен (повторите шаги выше)
